Italics
    Toc


surround the word with underscore or singel asterisk(_/*) e.g:  


_this is a test using underscore_  



 this is a test using underscore
 this is a test using asterisks  






bold
    Toc 


surround words with two asterisks(**) e.g:  


**this is a test**  



this is a test 






bold and italics
     Toc 


of course, words can be both italics and bold, no matter which order of the asterisks and underscores. even can make a header bold and italics e.g: 


**_this is a test_**  



this is a test






links
    Toc 


surround keywords with brackets([]) and correspond url with parenthesis(( )) e.g:  


[link to markdown tutorials](https://www.markdowntutorial.com/lesson/3/)



link to markdown tutorials 




and another type of links is reference link. the syntax goes as this : 





this is the example for reference links(link to google search)
this is another example, same links to google search




the advantage of using reference link rather than regular link is that when multiple place have the same link and a change need to be done. user can simply change the reference link rather than change several links in different place.




block quotes
    Toc 


preface a link with “greater than” caret e.g:


> "quotes goes here"  



“quotes goes here”




if the quote span multiple paragraph, use caret on each link of quotes.
